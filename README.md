# 🚀 Quick Start

## Environment Setup

**Clone the Repository**
```bash
git clone https://github.com/KimotaQY/SAM2-CD.git
cd SAM2-CD
```

**Install Dependencies**
```bash
pip install -r requirements.txt
```

## Run an Example
```bash
python train.py
```
